◆ menu_global()

void menu_global ( )

The global menu for the game.

This function serves as the central hub for the game, allowing the user to:

  • Play the game,
  • Create boats and place them on the board,
  • Shoot at the enemy,
  • Make the computer take its turn,
  • Declare the winner.

It manages game flow, including initializing game structures, taking user input for game options, and handling the main game loop until a winner is declared.
